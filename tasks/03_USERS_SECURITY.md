# 03) Users, security, and portal groundwork

## 3.1 Internal users

- [ ] Define groups/menus per department.
- [ ] Confirm audit log visibility rules.

Acceptance:
- Users see only what they need.

## 3.2 Portal users (mechanics) — groundwork

- [ ] Decide whether mechanics are:
  - portal users (recommended), or
  - internal users (usually not recommended)
- [ ] Define what mechanics can see:
  - their own orders/invoices/balance
  - product search (optional)

Acceptance:
- You can create a portal user and they can log in.
